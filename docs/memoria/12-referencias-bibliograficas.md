Para el desarrollo de este proyecto se han consultado las siguientes fuentes técnicas y estándares:

## Generales y de Arquitectura Distribuida

* **XDG Base Directory Specification:** Estándar de FreeDesktop para la organización limpia de archivos de configuración en sistemas tipo UNIX. [https://specifications.freedesktop.org/basedir-spec/latest/](https://specifications.freedesktop.org/basedir-spec/latest/)
* **RFC 8259 - The JavaScript Object Notation (JSON) Data Interchange Format:** Especificación del estándar para el intercambio de datos estructurados entre el cliente Python y el servidor Spring Boot. [https://datatracker.ietf.org/doc/html/rfc8259](https://datatracker.ietf.org/doc/html/rfc8259)

## Específicas del Cliente Python

* **Python Software Foundation:** Documentación oficial de Python 3.13. [https://docs.python.org/3/](https://docs.python.org/3/)
* **Qt for Python (PySide6):** Manual de referencia oficial de la librería gráfica, gestión de eventos y componentes `QWidget`. [https://doc.qt.io/qtforpython/](https://doc.qt.io/qtforpython/)
* **Git SCM Bare Repositories:** Documentación de Git sobre el uso de repositorios no de trabajo (*Bare*) para la gestión de dotfiles. [https://gitscripts.com/create-bare-repository-git](https://gitscripts.com/create-bare-repository-git)
* **SQLAlchemy ORM:** Referencia completa de la API del ORM de Python para el modelado relacional offline. [https://www.sqlalchemy.org/](https://www.sqlalchemy.org/)
* **SQLite Database Engine:** Documentación del motor relacional integrado utilizado en el cliente local. [https://www.sqlite.org/docs.html](https://www.sqlite.org/docs.html)
* **pytest:** Framework de pruebas unitarias y de integración para asegurar la calidad del cliente. [https://docs.pytest.org/](https://docs.pytest.org/)
* **Rich Library:** Librería para el renderizado avanzado de terminales y formateo semántico de la CLI. [https://rich.readthedocs.io/](https://rich.readthedocs.io/)

## Específicas del Servidor Java

* **Spring Boot Framework:** Guía de referencia y documentación de desarrollo del framework para microservicios. [https://spring.io/projects/spring-boot](https://spring.io/projects/spring-boot)
* **Spring Data JPA / Hibernate:** Especificación de persistencia relacional y mapeo objeto-relacional (ORM) para Java. [https://spring.io/projects/spring-data-jpa](https://spring.io/projects/spring-data-jpa)
* **Google Gemini API (GenAI SDK):** Manual de integración de modelos fundacionales, Batch API e inferencia mediante JSON estructurado. [https://ai.google.dev/gemini-api/docs](https://ai.google.dev/gemini-api/docs)
* **Apache Maven:** Sistema de gestión de proyectos y ciclo de vida de compilación para el servidor Java. [https://maven.apache.org/](https://maven.apache.org/)
