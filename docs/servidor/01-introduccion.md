El servidor de **Dotfiles-Manager** constituye el cerebro central y la base de conocimiento global del sistema. Desarrollado con **Java** y **Spring Boot**, actúa como un "oráculo" que centraliza la inteligencia necesaria para clasificar configuraciones de software que los clientes locales no pueden identificar por sí mismos.

Su principal responsabilidad es ofrecer una API REST eficiente y segura que gestione una cola de tareas asíncronas, delegando el análisis complejo a modelos de Inteligencia Artificial y persistiendo los resultados para que el conocimiento sea compartido por toda la comunidad de usuarios.
