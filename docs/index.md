## :material-information-outline: Introducción

**Dotfiles-Manager** es una solución integral para la gestión de archivos de configuración (*dotfiles*) en Linux. El sistema utiliza una arquitectura distribuida para permitir el escaneo local, la clasificación inteligente mediante IA y el almacenamiento centralizado de configuraciones.

Este repositorio está organizado como una **Wiki Técnica**.
