## 1. Introducción al módulo cliente

El módulo cliente de **Dotfiles-Manager** es el componente encargado de la interacción directa con el sistema operativo del usuario. Desarrollado íntegramente en **Python**, su objetivo es auditar el directorio personal ($HOME), identificar archivos de configuración y gestionar su ciclo de vida mediante control de versiones.

Este módulo ha sido diseñado bajo los principios de **Zero-Friction**, asegurando que el usuario no tenga que modificar sus hábitos de trabajo ni mover sus archivos de sitio para mantenerlos bajo control. El cliente actúa como un orquestador que combina herramientas de sistema (Git), bases de datos locales (SQLite) y servicios remotos (API Java) para ofrecer una experiencia de usuario fluida y segura.
