Índice General del Proyecto

Bienvenido a la documentación oficial y técnica del Trabajo de Fin de Grado: Dotfiles-Manager. Este documento web interactivo recoge el proceso íntegro de diseño, análisis y desarrollo arquitectónico del sistema.

Cómo usar esta documentación

Esta memoria está diseñada como un documento web interactivo de lectura progresiva. Para facilitar la navegación por secciones técnicas extensas, se ha dividido en páginas individuales por subsección en lugar de un único documento monolítico.

Menú lateral

El panel izquierdo muestra el índice completo de la memoria. Las secciones que contienen subsecciones muestran un submenú desplegable con las páginas individuales (por ejemplo, 4.1 Topología, 4.2 Flujo de Datos, etc.). El ítem activo aparece resaltado en verde para indicar la página actual en todo momento.

Botones de navegación

La cabecera de cada página incluye dos botones:

  • ← Anterior — navega a la subsección inmediatamente anterior en el orden de lectura.
  • Siguiente → — avanza a la siguiente subsección.

Estos botones permiten recorrer la memoria de principio a fin de forma secuencial, siguiendo el hilo narrativo del proyecto, sin necesidad de volver al índice entre cada sección.
