7.6. Vista: Ajustes

Figura 9: Vista Ajustes.

La vista de Ajustes centraliza la configuración del comportamiento interno de la herramienta. Su objetivo es ofrecer al usuario control total sobre los parámetros de la aplicación sin necesidad de editar archivos de configuración manualmente.

Funcionalidades Planificadas

Configuración del Servidor
  • Endpoint del Oráculo: campo editable con la URL del servidor central al que la aplicación consultará para identificar archivos desconocidos vía IA.
  • Verificar conexión: botón para comprobar que el servidor es accesible desde la máquina actual (check_updates).
  • Fecha y estado de la última conexión exitosa con el servidor.
Gestor de Reglas de Ignorados
  • Listado de las reglas activas en global_ignore_rules (tipo REGEX, EXACT o EXTENSION).
  • Añadir nuevas reglas de exclusión personalizadas.
  • Activar o desactivar reglas sin eliminarlas.
  • Eliminar reglas existentes.
Mantenimiento
  • DB Wipe: opción para resetear completamente la base de datos local, devolviendo la aplicación a su estado inicial.
  • Exportar / Importar ajustes: serialización de la configuración actual para facilitar la migración a otros equipos.
