7.2.4 Administración y Mantenimiento

Submódulo destinado al control macro y a garantizar que el software local se mantenga cohesionado y sano.

Ajustes Generales
  • Gestor de Ignore Engine: Listado global tipo .gitignore editable en la interfaz (permitiendo añadir exclusiones como *.tmp o .*_history) previniendo que los ciclos iniciales lean siquiera los ficheros en disco.
  • Conexión Backend: Determinación manual de las URL del API Endpoint de modo que si se aloja de forma autogestionada, el sistema pueda interactuar.
  • Vaciado de estado local (DB Wipe) para purgar artefactos inconsistentes.

Nota: A medida que avance el alcance, está concebida la integración en este último bloque de módulos como las copias de seguridad estáticas tipo tar.gz.
