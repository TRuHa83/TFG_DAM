7.5. Vista: Archivos Desconocidos

La vista de Archivos Desconocidos lista todas las rutas de dotfiles que el motor heurístico local no ha podido asociar automáticamente a ninguna aplicación registrada en known_apps_reference. Esta vista es el punto de entrada para resolver manualmente —o mediante IA— las configuraciones huérfanas del sistema.

Figura 5: Vista Dotfiles Desconocidos.

Listado de Rutas sin Identificar

Cada entrada de la lista muestra:

Información de la ruta
  • Ruta absoluta del archivo o directorio de configuración detectado en el $HOME.
  • Tipo: archivo individual o directorio de configuración.

Acciones de Resolución

Opciones por cada ruta desconocida
  • Vincular a aplicación existente: asociar la ruta a una aplicación ya registrada en la base de datos local.
  • Crear nueva aplicación: registrar una nueva entrada en known_apps_reference y vincularle la ruta directamente.
  • Añadir a ignorados: crear una regla en global_ignore_rules para que esta ruta no vuelva a aparecer en futuros escaneos.
  • Consultar al LLM (futuro): enviar los metadatos de la ruta al servidor central para que un Modelo de Lenguaje intente identificar a qué software pertenece.

Nota: La visualización del listado está funcional. Las acciones de resolución (vincular, crear app, ignorar) están planificadas para la siguiente iteración del desarrollo.
